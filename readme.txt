== Description ==

WP Redactor allow users to redact content by using a blacklist of terms configured in the site settings or by manually redacting content in the editor using a custom menu item and short code.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-redactor` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin
